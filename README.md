## 介绍Serverless，无服务器并不是真的无服务器

   ### Serverless是什么
   * Serverless直译为中文是“无服务器”，但是实际上它仍需要服务器，只不过服务器的管理以及资源分配部分对用户不可见。
   * 从程序运行环境看，从最初的物理服务器进化到虚拟化提供的虚拟机，由虚拟机进化到容器，而Serverless 架构又提供一个比容器更轻量、更简单的环境。
